# Q8 Sport Market

منصة موجهة لسوق الكويت لبيع السيارات السبورت وقطع الغيار والمطلوبات المباشرة من المستخدمين.

المنتج المستهدف ليس تطبيق عرض فقط. الاتجاه الجديد هو منصة واحدة بهوية قوية تعمل على:

- تطبيق iOS
- تطبيق Android
- موقع ويب بنفس اللغة والهوية والبيانات

## الفكرة الأساسية

السوق في الكويت يتحرك بسرعة عندما تكون المطلوبات واضحة وسهلة الوصول. لذلك المنتج يعتمد على مسارين واضحين:

- `معروض`: سيارات سبورت وقطع غيار مع تواصل مباشر
- `مطلوب الآن`: المستخدم يكتب ما يحتاجه والسوق يتفاعل معه مباشرة

## ما تغير في هذا الريبو

- إزالة مسارات ومنطق الأدمن من الواجهة الأساسية
- تحويل الواجهة الرئيسية إلى سوق حي فيه معروضات ومطلوبات
- إضافة صفحة تفاصيل لقطع الغيار
- تقوية تجربة الحساب لتخدم المستخدم العادي لا الإدارة

## التشغيل

### React Native

```sh
npm start
npm run ios
npm run android
```

### iOS Pods

```sh
bundle install
cd ios && bundle exec pod install
```

## Firebase

المشروع يعتمد حاليًا على:

- Firebase Auth
- Realtime Database
- Firebase Storage

لازم يكون إعداد `DATABASE_URL` موجود بشكل صحيح داخل ملفات Firebase الأصلية للتطبيق.

### حذف تلقائي للإعلانات المباعة (3 ساعات)

تم إضافة Cloud Functions مجدولة تقوم بحذف إعلانات السيارات/القطع المعلّمة "مباع" بعد 3 ساعات تلقائياً، حتى لو ما أحد فتح التطبيق.

### حذف تلقائي للإعلانات القديمة (30 يوم)

تم إضافة Cloud Function مجدولة تقوم بحذف أي إعلان (سيارات/قطع/مطلوب) إذا مرّ عليه 30 يوم من آخر تحديث (`updatedAt` وبالرجوع إلى `createdAt` عند عدم وجوده).

المتطلبات:

- تفعيل Firebase Functions + Cloud Scheduler (عادة يتطلب خطة Blaze)

النشر:

```sh
cd functions
npm install
npm run build
firebase deploy --project <YOUR_FIREBASE_PROJECT_ID> --only functions,database
```

## الاتجاه التالي
