# Q8 Motors - تحديث النظام الكامل

## التغييرات الرئيسية

### 1. التصميم الجديد (أسود، أبيض، أحمر)
- تم تغيير جميع الألوان لتصبح:
  - الخلفية: أسود (#0a0a0a)
  - النصوص: أبيض (#ffffff)
  - اللون الأساسي: أحمر (#dc2626)
  
### 2. نظام السيارات والقطع
- **تصنيفات السيارات المدعومة:**
  - Ford: Mustang, F-150 Raptor, GT, Shelby GT500
  - Chevrolet: Corvette, Camaro, Silverado ZR2
  - Toyota: Supra, GR86, Tundra TRD Pro
  - Dodge: Challenger, Charger, Viper
  - Nissan: GT-R, 370Z, Titan
  - BMW: M3, M4, M5, M8
  - Mercedes: AMG GT, C63 AMG, E63 AMG
  - Porsche: 911, Cayman, Panamera

### 3. نظام الخصوصية
- **إخفاء اسم البائع:** يظهر فقط رقم الهاتف/واتساب
- **لا حاجة لإنشاء حساب** لرفع الإعلانات
- **التواصل المباشر** عبر الهاتف/واتساب

### 4. الصفحة الرئيسية
- عرض ديناميكي للسيارات وقطع الغيار
- فلتر متقدم حسب:
  - نوع المنتج (سيارة / قطعة غيار)
  - الماركة
  - الموديل
- تصميم احترافي بألوان داكنة

### 5. صفحة إضافة الإعلان (/add-listing)
- نموذج شامل لإضافة:
  - السيارات الكاملة مع كل التفاصيل
  - قطع الغيار
- رفع صور متعددة
- معلومات الاتصال (هاتف/واتساب)

### 6. قاعدة البيانات
تم إضافة الحقول التالية لجدول `products`:
```sql
contactPhone TEXT       -- رقم الهاتف
contactWhatsapp TEXT    -- رقم الواتساب
showSellerName BOOLEAN  -- إخفاء اسم البائع (افتراضي: false)
```

## الملفات المحدثة
- `src/app/page.tsx` - الصفحة الرئيسية الجديدة
- `src/app/globals.css` - الألوان والتصميم الجديد
- `src/app/add-listing/page.tsx` - صفحة إضافة الإعلان
- `prisma/schema.prisma` - تحديث نموذج البيانات

## كيفية الاستخدام

### تشغيل المشروع محلياً
```bash
npm install
npm run db:push
npm run db:generate
npm run dev
```

### إضافة إعلان جديد
1. اذهب إلى `/add-listing`
2. اختر نوع الإعلان (سيارة / قطعة غيار)
3. املأ التفاصيل
4. أضف الصور
5. أضف معلومات الاتصال
6. انشر الإعلان

### البحث والفلترة
- استخدم الفلاتر في الصفحة الرئيسية
- اختر نوع المنتج، الماركة، والموديل
- النتائج تظهر مباشرة

## نظام تسجيل المبيعات
- صاحب الإعلان يمكنه تسجيل المنتج كمباع
- API: `PATCH /api/products/[id]/status`
```json
{
  "status": "SOLD",
  "soldPrice": 15000
}
```
